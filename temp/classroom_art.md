아래 정보를 참고해서 아이용 틀린그림찾기 이미지를 만들어줘.

제목: Classroom Art
테마: classroom

틀린그림 설명 6개:
1. 칠판 위 별 그림 색이 달라졌어요.
2. 책상 위 크레용 하나가 사라졌어요.
3. 그림 종이에 그려진 해 모양이 달라요.
4. 가방에 달린 배지 모양이 바뀌었어요.
5. 물감 통의 색 배열이 달라졌어요.
6. 책 표지 색이 달라졌어요.

요구사항:
- 1:1 정사각형 이미지로 만들어줘.
- 왼쪽 그림과 오른쪽 그림을 한 이미지 안에 나란히 배치해줘.
- 왼쪽은 원본, 오른쪽은 틀린그림 버전이야.
- 전체 분위기는 제목과 테마만 참고하고, 틀린 부분은 위 6개 설명을 사용해줘.
- 틀린그림은 정확히 6개만 만들어줘.
- 위 설명에 없는 추가 차이는 만들지 마.
- 6개의 차이는 서로 충분히 떨어져 있어야 해.
- 각 차이는 아이가 봐도 명확하게 알아볼 수 있어야 해.
- 너무 작거나 애매한 차이는 만들지 마.
- 배경 질감, 그림자, 노이즈처럼 헷갈리는 차이는 만들지 마.
- 글자, 숫자, 로고, 워터마크는 넣지 마.
- UI 버튼이나 설명 텍스트도 넣지 마.
- 두 그림은 거의 동일해야 하며, 오직 6개의 명확한 차이만 있어야 해.
