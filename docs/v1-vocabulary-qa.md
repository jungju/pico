# Pico V1 Vocabulary QA

Use this checklist for every v1 stage before release.

## Stage Metadata

- Stage has a clear English title and Korean title.
- Theme and level match the visual/content difficulty.
- Estimated play time is suitable for a short child session.

## Word Checks

- English words are common, concrete, and age-appropriate.
- Korean meanings are short and natural for children.
- The word matches the visible object or card image.
- There are no duplicate words inside one target/pair list unless the game
  intentionally uses a matching pair.

## Pronunciation And Audio

- Phonetic text, when present, is correct enough for parent reference.
- Web Speech text uses natural English words or short sentences.
- Optional audio paths, when present, point to committed assets.
- Audio/speech does not include long explanations.

## Sentence Checks

- Example sentence uses the target word naturally.
- Sentence length is short enough to hear in one breath.
- Korean translation matches the sentence meaning.
- Sentence content is safe and child-friendly.

## Game-Type Specific Checks

- Spot the Difference: difference labels describe what changed.
- Hidden Objects: target words match visible scene objects.
- Maze: collectible words match the collectible icon or theme.
- Memory Cards: pair words match both card faces.
